    TUGAS 2 - Program Newton Raphson pada Fungsi Persamaan Non Linear¶

    Newton-Raphson¶

    Pengertian¶

    Newton-Raphson atau biasanya didefinisikan sebagai sebuah metode pencarian akar pada fungsi f(x) melalui pendekatan satu titik dengan fungsi f(x) yang memiliki turunan. Metode ini memakai pendekatan satu titik untuk titik awalnya. Oleh karena itu dianggap lebih mudah dari Bisection Method (Metode Bagi Dua). Convergen ke akar bisa semakin cepat dilakukan apabila titik awal yang dipilih dengan akar yang sebenarnya semakin dekat.

    Prosedur¶

    Untuk menentukan prosedur perlu menentukan x_0 sebagai titik awal, kemudian menarik garis lurus ( misal garis: imath ) yang menyinggung titik f(x_0). Hal ini berakibat garis imath memotong sumbu x di titik x_1. Ulangi langkah sebelumnya dengan x_1 sebagai titik awalnya. Setelah beberapa kali perulangan maka akan mendapatkan titik x_2, x_3, x_4, ... , x_n dengan x_n adalah bilangan ril yang merupakan akar atau mendekati akar yang sebenarnya.

    Atau bisa digambarkan seperti dibawah ini

    .

    .

    .

    Dari gambar diatas, maka kita bisa menurunkan rumus metode newton-raphson, sebagai berikut:

    persamaan garis imath: y - y_0 = m(x-x_0)

    y - f(x_0) = f'(x_0)(x - x_0)

    x_1 perpotongan garis imath dengan sumbu -x

    0 - f(x_0) = f'(x_0)(x - x_0)

    y = 0 dan x = x_1 maka koordinat titik x_1, 0

    - \frac{f(x_0)}{f'(x_0)} = (x_1 - x_0)

    sehingga didapatkan formula sebagai berikut:

    x_1 = x_0 - \frac{f(x_0)}{f'(x_0)} , x_2 = x_1 - \frac{f(x_1)}{f'(x_1)}, ... , x_n = x_{n-1} - \frac{f(x_{n-1})}{f'(x_{n-1})}

    .

    Algoritma¶

    Dari formula diatas, dapat disusun sebuah algoritma yang kemudian diimplementasikan menjadi sebuah program.

    Berikut adalah algoritma untuk menyusun akar - akar f(x) = 0 :

    1. Definisikan fungsi f dengan f(x) dan f'(x).

    2. Tentukan epsilon sebagai toleransi kesalahan serta iterasi maksimum untuk Stopping Condition.

    3. Pilih tebakan awal x_0.
    4. Hitung f(x_0) dan f'(x_0).
    5. Hitung x_b = x_0 - \frac{f(x_0)}{f'(x_0)}f'(x_0) \neq 0. Jika f'(x_0) = 0 kembali ke langkah 3.
    6. Jika \left | x_b - x_0 \right | < \varepsilon itersasi lebih dari iterasi maksimum tulis x_{hampiran} = x_b sebagai hasil hampiran akar. Jika tidak, lanjutkan ke langkah berikutnya.
    7. Ganti nilai x_0 dengan x_0 = x_b dan kembali ke langkah-4.

    Atau bisa juga menggunakan program python sebagai berikut.

    Program Python¶

    import math
e = 2.71828

def fungsi(x):
    x = float((e**x) - (4*x))
    return x

def fungsiturunan(x):
    x = float((e**x) - (4))
    return x

x = float(input('Masukkan nilai awal = '))
error = float(input('Masukkan nilai error = '))
perulangan = int(input('Masukkan maksimal pengulangan = '))

iterasi = 0
selisih = error+1
while iterasi <= perulangan  and selisih>error :
    iterasi += 1
    f_2 = x - (fungsi(x)/fungsiturunan(x))
    selisih = math.fabs(f_2 - x)
    x = f_2
    print("\niterasi ke = ",iterasi,"\nx = ",f_2, "\nf(",f_2,") = ",fungsi(f_2),"\nselisih = ",error)
    if iterasi <= perulangan:
        print("Perulangan Mencapai Batas Maksimal dengan hasil = ", f_2)
    else :
        print("Toleransi tidak terpenuhi")

    .

    Keterangan :

    1. Import Library math (matematika).
    2. Karena kita menggunakan contoh fungsi f(x) = e^x - 4x maka kita membuat sebuah fungsi yang sesuai, dan juga fungsi turunannya yaitu f'(x) = e^x - 4.
    3. Membuat sebuah inputan untuk X , Error / Epsilon , serta Maksimal perulangan untuk stopping condition.
    4. Mendeklarasikan iterasi = 0 untuk perulangan yang ke-0 dan akan ditambah setiap kali perulangan.
    5. Mendeklarasikan selisih untuk x_b - x_0 untuk perbandingan.
    6. Melakukan perulangan dengan kondisi iterasi kurang dari sama dengan inputan maksimal iterasi dan selisih lebih dari error / epsilon
    7. Menghitung x_b dengan rumus yang sudah didapatkan sebelumnya.
    8. Melakukan perbandingan jika mencapai nilai True maka toleransi tidak terpenuhi.
    9. Jika pengecekan selisih > error bernilai True maka toleransi akan terpenuhi dengan nilai error serta fungsi x pada iterasi ke-n.
